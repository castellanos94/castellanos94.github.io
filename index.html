Jemoa
Java Evolutionary Multi-Objective Algorithms.

Jemoa is a framework developed to solve multiobjective optimization problems with evolutionary algorithms. This framework contains the following problems: DTLZ suite, knapsack problem, PSP. It includes the algorithms NSGA-II, NSGA-III, GWO, MOGWO, IMOACOR and some methods for Multi-Criteria Decision Support (MCDA).
